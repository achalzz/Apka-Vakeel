## Design Tips for PPT:

1. **Color Scheme:**
   - Primary: Blue (#0ea5e9)
   - Accent: Green (for India focus)
   - Background: White/Light gray

2. **Fonts:**
   - Headings: Bold, Sans-serif
   - Body: Clean, readable
   - Include Hindi text where relevant

3. **Visuals:**
   - Use screenshots of the app
   - Infographics for statistics
   - Icons for features
   - Charts for impact data

4. **Consistency:**
   - Same color scheme throughout
   - Consistent font sizes
   - Aligned elements
   - Professional look
